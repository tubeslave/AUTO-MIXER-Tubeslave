# Голосовое управление пультом

Модуль голосового управления использует библиотеку Faster-Whisper для распознавания речи и управления микшером через голосовые команды.

## Установка

Библиотека `faster-whisper` уже добавлена в `requirements.txt`. Для установки выполните:

```bash
cd backend
pip install -r requirements.txt
```

При первой загрузке модели Whisper автоматически скачает модель `small` (около 500MB).

## Использование

### Запуск голосового управления

Голосовое управление запускается через WebSocket команды:

```json
{
  "type": "start_voice_control",
  "model_size": "small",
  "language": "ru"
}
```

Параметры:
- `model_size`: размер модели Whisper (`tiny`, `base`, `small`, `medium`, `large`). По умолчанию `small`
- `language`: язык распознавания (`ru` для русского, `en` для английского, или `null` для автоопределения)

### Остановка голосового управления

```json
{
  "type": "stop_voice_control"
}
```

### Проверка статуса

```json
{
  "type": "get_voice_control_status"
}
```

Ответ:
```json
{
  "type": "voice_control_status",
  "active": true
}
```

## Поддерживаемые голосовые команды

### Русский язык

- **Управление каналами:**
  - "канал 1" - установить фейдер канала 1
  - "канал 5" - установить фейдер канала 5
  - "гейн 3" - установить гейн канала 3

- **Управление снапшотами:**
  - "загрузить концерт" - загрузить снапшот с именем "концерт"
  - "снапшот репетиция" - загрузить снапшот "репетиция"

- **Управление громкостью:**
  - "громче 2" - увеличить громкость канала 2
  - "тише 4" - уменьшить громкость канала 4

- **Mute:**
  - "мут 1" - заглушить канал 1

### English

- **Channel control:**
  - "channel 1" - set fader for channel 1
  - "gain 3" - set gain for channel 3

- **Snapshot control:**
  - "load concert" - load snapshot named "concert"
  - "snapshot rehearsal" - load snapshot "rehearsal"

- **Volume control:**
  - "louder 2" - increase volume for channel 2
  - "quieter 4" - decrease volume for channel 4

- **Mute:**
  - "mute 1" - mute channel 1

## События

При распознавании команды сервер отправляет событие:

```json
{
  "type": "voice_command_executed",
  "command": "set_fader",
  "channel": 1,
  "value": 0.5
}
```

## Технические детали

- **Модель**: Faster-Whisper small (баланс между точностью и скоростью)
- **Частота дискретизации**: 16 kHz
- **Буфер**: 3 секунды аудио для обработки
- **VAD фильтр**: Включен для фильтрации тишины
- **Потоковая обработка**: Аудио обрабатывается в отдельном потоке

## Добавление пользовательских команд

Вы можете добавить свои команды программно:

```python
voice_control.add_command(
    keyword="моя_команда",
    pattern=r"моя_команда\s+(\d+)",
    action="my_action"
)
```

Затем обработайте команду в методе `_execute_voice_command` в `server.py`.

## Требования

- Микрофон должен быть подключен и доступен системе
- PyAudio должен быть правильно установлен (может потребоваться установка системных библиотек)
- Для GPU ускорения установите CUDA и используйте `device="cuda"` при инициализации

## Известные ограничения

- Команды распознаются с задержкой ~3 секунды (размер буфера)
- Точность распознавания зависит от качества микрофона и окружающего шума
- Модель `small` требует около 2GB RAM для работы
