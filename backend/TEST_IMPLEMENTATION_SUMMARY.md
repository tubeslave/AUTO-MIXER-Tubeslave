## Использование

### Базовый тест:
```bash
# Терминал 1: Запуск backend
cd backend
python server.py

# Терминал 2: Запуск теста
cd backend
python test_gain_staging_dante.py --duration 10.0
```

### Тест с Wing:
```bash
python test_gain_staging_dante.py --wing-ip 192.168.1.100 --duration 10.0
```

### Анализ результатов:
```bash
python analyze_test_results.py test_results/gain_staging_dante_YYYYMMDD_HHMMSS.json
```

## Примечания

- Тест требует запущенный backend сервер
- Для полного тестирования требуется Dante устройство или Dante Virtual Soundcard
- Для применения корректировок требуется подключение к пульту Wing
- Минимальная длительность анализа для LUFS: 5 секунд
- Рекомендуемая длительность: 10-15 секунд для стабильных измерений
